Welcome to TLD to SMS

TldToSms is a python package which is designed to help you with being notified on new TLD as soon as it becomes public. How plugin notifies you is by sending SMS message to your mobile phone.

Requirements

  • You need to have TelAPI account
  • You must install package ( obviously lol ) Python 2.6+

By GitHub clone

$ cd ~
$ git clone https://github.com/0x19/tldtosms.git
$ cd tldtosms
$ python setup.py install

By PIP
